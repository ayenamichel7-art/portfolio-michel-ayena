## 🛠️ Stack Technique

Ce portfolio est sciemment gardé pur et minimaliste ("Vanilla") pour garantir des temps de chargement ultra-rapides et une sécurité de premier plan, sans s'appuyer sur des frameworks front-end lourds superflus.

- **HTML5 Sémantique**
- **CSS3 (Custom Properties & Animations)**
- **JavaScript ES6+ (Vanilla)**
- **Vercel Config** (en-têtes de sécurité "nosniff", "X-Frame-Options" via `vercel.json`)

## 🚀 Installation & Déploiement Local

Aucune étape d'installation (type `npm install`) n'est requise.

1. **Cloner le répertoire**
   ```bash
   git clone https://github.com/ayenamichel7-art/portfolio-michel-ayena.git
   cd portfolio-michel-ayena
   ```

2. **Visualisation Locale**
   Vous pouvez ouvrir le fichier `index.html` directement dans votre navigateur ou utiliser une extension type **Live Server** sur VS Code pour avoir du rechargement à chaud.

3. **Génération PDF du CV**
   Le CV est construit de telle manière (CSS print-media) qu'il tient parfaitement sur une feuille A4 avec Edge ou Google Chrome (Option: "Imprimer" > "Enregistrer au format PDF" > échelle 100%, marges nulles).

## 🌍 Déploiement

Ce dépôt est configuré pour être déployé instantanément sur **Vercel** grâce au fichier `vercel.json`. Il suffit d'associer ce dépôt à Vercel via leur interface.
